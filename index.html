Basics of the Web and HTML

The Web

The Web was invented around 1990's and approximately has 30 billion pages today. Whether it is a Document or an image or a video on the web they are linked or glued together through HTML.The Web is a collection of many HTML documents.When a person uses the internet through his browser , a request is placed to the server through HTTP (Hypertext Transfer Protocol) . Then the Server finds that particular HTML document and sends it to the user through the same channel.Here is a video that explains the concept more clearly.

HTML

HTML stands for Hyper text Markup Language. HTML is made up of text content (what you see ), markup (what it looks like), links to other pages and reference to other documents (like videos or images ).

  1. HTML Markup

    2. HTML Markups are set of commands inserted in a file for a particular display on the browser. They are made up of elements and tags. An example of basic tag is explained in this particular video. Tags are single commands placed between brackets whereas elements are components of HTML document. This video explains it in detail.

  2. HTML Attributes

    3. When a user while writing a HTML Document wants to refer a particular link on the web in this particular document, he uses the HTML attributes for the tag which in this case is an anchor tag. A clear example is given in this video .

  3. Inline (vs) Block elements

    4. Both these elements are used to organise the content. But there is a difference in the usage. Inline elements can display the organised content and span the space of the content. Whereas block elements place an invisible block around the content allowing it to be displayed as a new line. Unlike inline elements, block elements occupy the width of the parent if no width is specified. Here is a video thats explains this concept.

  4. Whitespace

    5. While creating HTML document in order to add spaces between the lines or to create a new paragraph, there must be a specific command in the text. otherwise, all the spaces given through tab are deleted while displaying the content. The additional space is called the whitespace
